Портфолио аналитики данных

Основной фокус: поиск инсайтов и объяснение поведения пользователей через данные

В портфолио представлены проекты анализа интернет-магазина, 
направленные на выявление факторов роста выручки, 
поведения клиентов и проблем с удержанием.


Во всех проектах:
  
- рассчитаны ключевые метрики
- проведён анализ данных
- сделаны бизнес-выводы и рекомендации
  
 Навыки:

- SQL (PostgreSQL): JOIN, GROUP BY, CTE, оконные функции
- Анализ данных: метрики, сегментация, когортный анализ
- Работа с данными: очистка, агрегация, подготовка
- Визуализация: Excel / Power BI
