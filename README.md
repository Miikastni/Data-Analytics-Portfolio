# Data-Analytics-Portfolio

## Обзор

Это портфолио демонстрирует навыки анализа данных на одном реальном датасете онлайн-ритейла. Каждый проект показывает разные компетенции:

- **RFM-анализ** — сегментация клиентов и оценка ценности  
- **Выручка и продуктовые метрики** — анализ продаж, средний чек, сезонность  
- **Cohort Retention** — удержание клиентов и когортный анализ  

Все проекты включают SQL-запросы, интерпретацию данных и рекомендации для бизнеса.

---

## Проекты

### 1. RFM-анализ
- **Описание:** сегментация клиентов по показателям Recency, Frequency, Monetary (RFM).  
- **Результат:** выявлены топ клиенты, проблемные клиенты, средние показатели по сегментам.  
- **Ссылка на репозиторий:** https://github.com/Miikastni/rfm-customer-segmentation.git

### 2. Анализ продаж и продуктовых метрик
- **Описание:** анализ выручки по месяцам, средний чек, количество заказов, сезонные пики и просадки.  
- **Результат:** выявлены ключевые месяцы с рекордной выручкой и причины (количество заказов vs средний чек).  
- **Ссылка на репозиторий:** https://github.com/Miikastni/online-retail-analysis.git

### 3. Cohort Retention
- **Описание:** когортный анализ удержания клиентов по месяцам с расчетом retention rate.  
- **Результат:** выявлены динамика удержания, лучшие и слабые когорты, рекомендации по улучшению удержания.  
- **Ссылка на репозиторий:** https://github.com/Miikastni/cohort-retention-analysis.git
